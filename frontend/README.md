# VectorShift — Frontend

The React + ReactFlow + Zustand client for the VectorShift pipeline builder.

See the [root README](../README.md) for the full project overview, architecture
notes, and setup instructions.

## Quick start

```bash
npm install
npm start
```

Runs the app in development mode at [http://localhost:3000](http://localhost:3000).
It expects the backend running at `http://localhost:8000` (override with
`REACT_APP_API_BASE`).

## Scripts

- `npm start` — run the dev server
- `npm run build` — production build into `build/`
- `npm test` — run tests
